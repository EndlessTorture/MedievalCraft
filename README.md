## Че это ?
Этот проект - попытка создать более менее нормальную игру, используя только нейросети

Если подробнее, то это клон майнкрафта с уклоном в средневековый стиль

### Приколы:
- Процедурная генерация текстур и звуков, никаких ассетов
- Чистый WebGL
- Система партиклов

Надеемся, что из этого выйдет что-то нормальное...
